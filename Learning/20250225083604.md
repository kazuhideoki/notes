## 4.4 問題を「解く」ことの形式的な定義

判定問題

```
判定問題とは出力が bool の関数と理解していい
```


P(I) ∈ F(I)

```
入力（I）に対して
	•	その正しい解答の集合（F(I)）がある
	•	プログラム（P）が入力 I を受け取って出力したもの（P(I)）がその正解集合に含まれていれば、問題を解いているとみなす、
っていうふうに考えているわけだね
```
